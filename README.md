# PreSchool Management System

A modern PreSchool Management System built with:

- .NET 10
- Clean Architecture
- Vertical Slice Architecture
- CQRS + MediatR
- Entity Framework Core
- SQL Server
- AutoMapper
- FluentValidation
- JWT Authentication
- Serilog
- React + TypeScript
- Scalar

## Architecture

- Domain - Entities and DTOs
- Application - CQRS
- Infrastructure - Business Layer
- Persistence - Application DB Context and DB related objects
- API - Actual API
- Shared - Common methods,static variables
